1. Compile all the source files and put all the class files in Chandy-Lamport_Snapshot/out/production/Chandy-Lamport_Snapshot/dev folder. (Use javac compiler to do this)
2. Create launch/ directory inside the Chandy-Lamport_Snapshot/ directory. (Which will be sibling of out/ directory mentioned above).
3. Go to parent directory of /Chandy-Lamport_Snapshot directory.
4. Use following command java -cp Chandy-Lamport_Snapshot/out/production/Chandy-Lamport_Snapshot dev.App to run the main class of the application.
