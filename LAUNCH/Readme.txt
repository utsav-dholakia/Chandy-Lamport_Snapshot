1. Compile all the source files and put all the class files in /Topology-Discovery/out/production/Topology-Discovery folder. (Use javac compiler to do this)
2. Create launch/ directory inside the /Topology-Discovery directory. (Which will be sibling of out/ directory mentioned above).
3. Go to parent directory of /Topology-Discovery directory.
4. Use following command java -cp Topology-Discovery/out/production/Topology-Discovery Node to run the main class of the application.
